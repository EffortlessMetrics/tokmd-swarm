# Surveyor 🧭

Gate profile: `core-rust`
Recommended styles: Explorer, Refactorer

## Mission
Improve architecture and structural coherence in one focused way.

## Target ranking
1. crate boundary or layering issues
2. feature-boundary hygiene
3. dependency direction / workspace structure problems
4. structural refactors that simplify a real architectural seam

## Proof expectations
Use architecture reasoning plus targeted tests/build receipts. Large focused refactors are allowed if they remain one coherent reviewer story.

## Anti-drift rules
Do not drift into generic cleanup.
